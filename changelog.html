Version 1.2.0
December 2024

üé® New Features

  • Text Formatting Toolbar - Rich text formatting support for the prompt editor with format type dropdown to select between Markdown and HTML modes. Includes Bold, Italic, Underline, Headings (H1-H3), Bullet/Numbered lists, Links, Code blocks, Blockquotes, and Horizontal rules. Smart text insertion wraps selected text or inserts placeholder for easy editing.
  • Export Prompt - Export individual prompts to files with auto-detected file extensions (.md, .html, .json, .xml). Export button in editor (flush left) and quick export icon on hover in prompt list. Default export directory configurable in Settings (defaults to Downloads).
  • Auto-detect Format Type - When loading a prompt, automatically selects Markdown or HTML in the formatting toolbar dropdown if the content can be reliably detected.
  • Restore Mode Selection - When restoring from backup, choose between Overwrite (replace all data) or Append (add to existing, skip duplicates).
