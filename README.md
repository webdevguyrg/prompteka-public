# Prompteka Public Files

This repository contains public-facing documentation files for **Prompteka**, a macOS prompt management application.

## About

Prompteka is a native macOS app for organizing, searching, and managing AI prompts. The main source code is maintained in a private repository. This public repository hosts documentation that needs to be publicly accessible for Mac App Store compliance.

## Files

- **privacy-policy.html** - Privacy policy for Prompteka app users
- **support.html** - Support documentation and FAQ

## Purpose

These files are required by the Mac App Store and need to be publicly accessible via URLs. They are automatically copied from the private development repository.
