How It Works

üîë Standard Protocol

Prompteka implements the Model Context Protocol (MCP)‚Äîa standardized specification by Anthropic for connecting LLMs and tools. This means:

  • Works with any MCP-compatible client (Claude, other LLMs, CLI tools)
  • No vendor lock-in or custom integrations needed
  • Language and platform agnostic

‚öôÔ∏è Persistent Service

Unlike simple hooks or scripts, the Prompteka MCP server runs continuously with:

  • Persistent SQLite database connection
  • Transaction-based consistency
  • Automatic retry and error recovery
  • WAL (Write-Ahead Logging) for safe concurrent access

üõ†Ô∏è Full CRUD Operations

Complete control over your prompt library:

  • Create: New prompts with metadata (emoji, color, folders)
  • Read: Search, filter, and retrieve prompts instantly
  • Update: Modify any prompt property atomically
  • Delete: Remove prompts or entire folder hierarchies safely

üì¶ Backup & Restore

Import/export your entire prompt library with full hierarchy preservation:

  • Export as JSON for version control
  • Backup and restore with overwrite options
  • Folder structure and metadata fully preserved
  • Perfect for migration, archiving, or disaster recovery
